# FlightMill_Project (What)

##overview (Why)
This project analyzes flight ability of bunny bees using flight mill and arduino uno.
flight mill과 아두이노 uno를 이용해 꿀벌의 비행 능력을 측정하는 과제입니다.

## hardware (How)
-Flight mill made by a 3D printer
-IR sensor (reflective)
-r is 122.5mm (it can be changed)
-use neodymium magnet to reduce the friction

## software (How)
-Arduino IDE (C++)

-Institute of Agricultural Science, CNU
